# Preset Buttons

## Purpose
Add preset buttons for common egg times (e.g., soft, medium, hard boiled).

## Tasks
- Add buttons for 3, 5, 7 minutes (or configurable)
- Integrate with timer logic
- Clearly label each preset

## Acceptance Criteria
- User can start timer with a single click for common presets
- Presets are clearly labeled and easy to use
